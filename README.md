## 在项目安装
```
npm install code-ui-vue -S
```

### 在项目全局注册
```
import Vue from "vue"
import CodeUI from "code-ui-vue"

Vue.use(CodeUI) // 这样你就可以开始使用CodUI的相关内容了
```
